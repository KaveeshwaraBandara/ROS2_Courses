# ROS 2 Nav2 with SLAM and Navigation

This workspace provides robot description and bringup content used for navigation-related exercises. It includes a Gazebo world, launch files, and a robot model with sensor components.

## Packages

- `my_robot_description`: URDF/Xacro files (including lidar and camera) and RViz configuration.
- `my_robot_bringup`: Launch files and a Gazebo world for simulation.

## Key Files

- `src/my_robot_description/urdf/`: Robot model sources (URDF/Xacro), including `lidar.xacro`.
- `src/my_robot_description/rviz/mobile_robot_config.rviz`: RViz display configuration.
- `src/my_robot_bringup/worlds/my_world.sdf`: Gazebo world.
- `src/my_robot_bringup/launch/`: Gazebo and display launch files.

## Build

```bash
colcon build
source install/setup.bash
```

## Run (Examples)

```bash
ros2 launch my_robot_bringup my_robot_gazebo.launch.xml
```

```bash
ros2 launch my_robot_bringup arm_gazebo_launch.xml
```

```bash
ros2 launch my_robot_description display.launch.py
```

```bash
ros2 launch my_robot_description arm_display.launch.xml
```

## Notes

- `build/`, `install/`, and `log/` are generated by colcon.
- Update Xacro files under `src/my_robot_description/urdf/` to change sensors or geometry.
