# ROS 2 Level 3: Advanced Concepts

This workspace collects advanced ROS 2 examples: custom interfaces, actions in C++/Python, and lifecycle nodes in both languages.

## Packages

- `my_robot_interfaces`: Custom action interface `CountUntil`.
- `actions_cpp`: Action client/server examples in C++.
- `actions_py`: Action client/server examples in Python.
- `lifecycle_cpp`: Lifecycle node example in C++.
- `lifecycle_py`: Lifecycle node example in Python.
- `my_robot_bringup`: Launch files for lifecycle testing.

## Key Files

- `src/my_robot_interfaces/action/CountUntil.action`: Custom action definition.
- `src/actions_cpp/src/`: C++ action client/server sources.
- `src/actions_py/actions_py/`: Python action client/server sources.
- `src/lifecycle_cpp/src/number_publisher.cpp`: C++ lifecycle example.
- `src/lifecycle_py/lifecycle_py/number_publisher.py`: Python lifecycle example.
- `src/my_robot_bringup/launch/`: Lifecycle test launch files.

## Build

```bash
colcon build
source install/setup.bash
```

## Run (Examples)

```bash
ros2 launch my_robot_bringup lifecycle_test.launch.py
```

```bash
ros2 run lifecycle_cpp number_publisher
```

```bash
ros2 run lifecycle_py lifecycle_node_manager
```

```bash
ros2 run actions_cpp count_until_server
```

```bash
ros2 run actions_cpp count_until_client
```

```bash
ros2 run actions_py count_until_server
```

```bash
ros2 run actions_py count_until_client
```

## Notes

- Run C++ and Python nodes from their respective packages after building.
- `build/`, `install/`, and `log/` are generated by colcon.
